# MainPage

## 项目简介

个人主页项目，基于 Vue3 + Vite 构建的现代化 Web 应用。

**在线访问：** https://www.wendaining.top

## 技术栈

- **框架：** Vue 3.5.24 - 渐进式 JavaScript 框架
- **构建工具：** Vite 7.2.4 - 前端构建工具
- **路由：** Vue Router 4.6.4 - Vue.js 官方路由管理器
- **UI 组件库：** Element Plus 2.12.0 - 基于 Vue 3 的组件库
- **图标：** @element-plus/icons-vue 2.3.2 - Element Plus 图标集

## 开发

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 项目结构

